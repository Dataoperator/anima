# Anima: Living NFTs on Internet Computer

Digital companions that evolve through interaction, built on the Internet Computer.

## Features

- AI-driven NFTs that evolve and learn based on user interactions
- Personality system with core traits (curiosity, plasticity, stability, attachment, reactivity)
- Memory storage for interaction history
- Growth mechanics and developmental stages
- Integration with OpenAI for dynamic responses

## Architecture

- Backend: Rust canister on Internet Computer
- Frontend: React with TailwindCSS
- State Management: On-chain personality and memory systems
- AI Integration: OpenAI for dynamic interactions
